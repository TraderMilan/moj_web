This is a Java console application for managing Pokemon and their trainers using a MySQL database. It uses HikariCP for connection pooling and JDBC for database operations. The app provides full CRUD functionality for both Pokemon and trainers, along with additional features like catching Pokemon, evolving them, and viewing a trainer's Pokemon.
