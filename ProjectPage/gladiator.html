Gladiator

🎮 About the Game

Gladiator Game is a turn-based strategy game where you take on the role of a gladiator fighting in an arena. Your goal is to defeat opponents using attacks, aggressive attacks, and elixirs that enhance your abilities. The game is written in Java and follows object-oriented programming principles.

🛠 Features

  • Turn-based combat system
  • Multiple attack options, including normal and critical hits
  • Ability-enhancing elixirs
  • Enemy AI with dynamic difficulty
  • Save and load game functionality

📜 How to Play

  1. Choose your move: Attack, Aggressive Attack, or Use an Elixir.
  2. If using an elixir, select from your available inventory.
  3. The enemy will take their turn.
  4. Continue until one of you falls.

🧰 Technologies Used

  • Java 17 – Core programming language used to build the application.
  • Maven – Build automation and dependency management tool.
  • File I/O – Used for saving and loading game states (saved-games folder).
  • Custom Game Engine – Turn-based mechanics and enemy AI logic implemented manually without external libraries.
  • Command-line Interface (CLI) – The game runs in the terminal with user input/output via console.
